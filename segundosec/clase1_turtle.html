¿Qué vamos a aprender hoy?

Turtle es una forma visual de empezar a programar: en vez de ver solo texto, vemos una tortuga que se mueve por la pantalla y dibuja según nuestras instrucciones.

➡️

Mover

forward()

↩️

Girar

left() / right()

✏️

Dibujar

Líneas y figuras

1. Importar

Primero activamos el módulo.

from turtle import *

2. Mostrar

Podemos ver a la tortuga en pantalla.

shape('turtle')

3. Avanzar

Se mueve la cantidad de pasos indicada.

forward(100)

4. Girar

Los giros se expresan en grados.

right(90)

Idea clave

Programar con Turtle es darle órdenes exactas a un personaje. Si una instrucción está mal escrita, Python no la entiende. Si está bien escrita, la tortuga la ejecuta.

Correcto

forward(80)

Error común

Forward 80

Recordá: Python distingue mayúsculas, paréntesis y comillas.

Cómo se mueve la tortuga

Primer programa completo

Este ejemplo dibuja una "L". Es ideal para entender el orden de las instrucciones.

from turtle import *
shape('turtle')
forward(120)
left(90)
forward(80)
`forward(120)` hace una línea recta.
`left(90)` cambia la dirección.
El último `forward(80)` dibuja la segunda parte.
Las órdenes se leen de arriba hacia abajo.

Antes de ejecutar

Abrí IDLE o el editor donde trabajen en clase.

Creá un archivo nuevo.

Copiá el código respetando mayúsculas y paréntesis.

Guardá el archivo.

Ejecutá con F5.

Si no aparece una tortuga sino una flecha, no está mal. `shape('turtle')` solo cambia el dibujo del cursor.

Figuras guiadas

🟦

Cuadrado

Un cuadrado tiene 4 lados iguales y se gira 90° en cada esquina.

from turtle import *
shape('turtle')
forward(100)
right(90)
forward(100)
right(90)
forward(100)
right(90)
forward(100)
🔺

Triángulo

Para un triángulo equilátero usamos giros de 120°.

from turtle import *
shape('turtle')
forward(120)
left(120)
forward(120)
left(120)
forward(120)

Mini resumen

`from turtle import *` activa las herramientas.

`shape('turtle')` muestra la tortuga.

`forward()` mueve hacia adelante.

`left()` y `right()` cambian la dirección.

El orden de las instrucciones importa.
