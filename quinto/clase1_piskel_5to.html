¿Qué es Piskel?

Piskel es una app para crear dibujos con píxeles. Un píxel es un cuadradito de color. Muchos cuadraditos juntos forman personajes, objetos y animaciones.

Cómo entrar a Piskel

Abrí el navegador.

Escribí `www.piskelapp.com`.

Tocá `Create Sprite` o `Create New Sprite`.

Elegí un tamaño simple, por ejemplo 32 x 32.

Ya podés empezar a dibujar.

Herramientas principales

Lápiz: para pintar píxeles.

Goma: para borrar.

Balde: para rellenar una zona.

Selector de color: para cambiar color.

Frames: para hacer animación.

Consejos

Empezá con dibujos chicos y fáciles.

No uses demasiados colores al principio.

Cambiá solo un detalle por frame.

Guardá o descargá tu trabajo al final.
