1. ¿Qué hace un sistema operativo?

🧠

Administra el hardware

Organiza qué hace el procesador, cuándo se guarda un archivo, cómo se usa la memoria RAM y qué periféricos responden.

🪟

Muestra una interfaz

Nos permite usar íconos, ventanas, botones, menús y toques en pantalla en lugar de escribir puro código.

🚀

Ejecuta aplicaciones

Hace posible abrir juegos, programas de dibujo, navegadores, cámaras, calculadoras y muchas otras apps.

🗂️

Organiza archivos y usuarios

Decide dónde se guardan los trabajos, qué permisos tiene cada persona y cómo se recupera la información.

3. El sistema operativo como organizador

Arrastrá cada tarjeta al lugar correcto. El objetivo es distinguir qué tareas resuelve el sistema operativo y cuáles dependen directamente del hardware.

Instrucción

Tocá y arrastrá cada tarjeta. Si una tarea habla de ventanas, archivos o apps, suele ir con el sistema operativo. Si habla de calcular, mostrar o guardar físicamente datos, pensá en el hardware.
