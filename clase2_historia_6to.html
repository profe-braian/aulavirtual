🖥️ Clase 2 • Hardware y Software

Como trabajan juntos por dentro

Ya sabemos que teclado es hardware y que los programas son software. Ahora subimos un nivel: veremos como se conectan, quien da las ordenes y para que sirve el sistema operativo.

⌨️ Hardware 💻 Software 🪟 Sistema operativo 🔁 Interaccion

Objetivo

Comprender como hardware, software y sistema operativo trabajan juntos para que una computadora funcione.

Idea clave

Ninguno trabaja solo: las partes fisicas necesitan instrucciones, y los programas necesitan un equipo que las ejecute.

⬅️ Volver al submenu
⌨️

Hardware

Es la parte fisica: lo que se puede tocar, conectar, mover o reparar.

💿

Software

Son programas e instrucciones que le dicen al equipo que hacer.

🪟

Sistema operativo

Organiza el trabajo entre programas, memoria, archivos y dispositivos.

🔁

Trabajo en equipo

El usuario da una accion, el software interpreta y el hardware responde.

Que hace el software

Lectura guiada

Cuando abres un programa, el sistema operativo le da espacio en memoria y le permite usar teclado, pantalla o parlantes.
Si no hubiera sistema operativo, usar varios programas y dispositivos seria mucho mas dificil.
Por eso decimos que el sistema operativo conecta al usuario, los programas y el hardware.
✅

Verdadero o falso

1. El monitor es hardware.

2. Un navegador web es hardware.

3. El sistema operativo ayuda a que los programas usen los dispositivos.

4. Hardware y software pueden trabajar totalmente separados.

Marca V o F en cada afirmacion y luego revisa.
